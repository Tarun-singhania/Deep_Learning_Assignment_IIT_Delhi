## 🚀 How to Use This Repository

Clone the repository:

```bash
git clone https://github.com/tarun-singhania/Deep_Learning_Assignment_IIT_Delhi.git
```

Move inside the repository:

```bash
cd Deep_Learning_Assignment_IIT_Delhi
```

Open any assignment folder:

```bash
cd Assignment_1
```

or

```bash
cd Assignment_2
```

or

```bash
cd Assignment_3
```

Each assignment folder contains its own README file with task-specific details.
